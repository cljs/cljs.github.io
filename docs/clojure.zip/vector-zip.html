Source docstring:
Returns a zipper for nested vectors, given a root vector
Source code @ clojurescript:src/main/cljs/clojure/zip.cljs
(defn vector-zip
  [root]
    (zipper vector?
            seq
            (fn [node children] (with-meta (vec children) (meta node)))
            root))
