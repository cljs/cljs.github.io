~@ unquote splicing

syntaxsince v0.0-1853 in clojureEdit

Like ~ unquote, except the result is spliced (i.e. [a b] => a b)

  • `(foo ~@x) => (cljs.user/foo a b) (if x is [a b])

Examples:

(def foo '[a b c])
`(~@foo)
;;=> (a b c)

(defn- read-unquote
  [rdr comma opts pending-forms]
  (if-let [ch (peek-char rdr)]
    (if (identical? \@ ch)
      ((wrapping-reader 'clojure.core/unquote-splicing) (doto rdr read-char) \@ opts pending-forms)
      ((wrapping-reader 'clojure.core/unquote) rdr \~ opts pending-forms))))
