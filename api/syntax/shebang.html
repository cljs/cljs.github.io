#! shebang

syntaxAdded in 0.0-927 in clojure

Equivalent to ; comment.

This is intended to allow executable ClojureScript files under Unix by allowing the first line of the file to have a shebang line).

Examples:

#!/bin/cljs
;; waits for another form since #!/bin/cljs was ignored.

123 #! this is ignored
;;=> 123
