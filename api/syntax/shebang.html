#! shebang

syntaxsince v0.0-1853 in clojureEdit

ClojureScript allows shebang lines simply by making #! equivalent to ; comment.

  • #!/usr/bin/env planck - use Planck to execute this file

Examples:

#!/bin/cljs
;; waits for another form since #!/bin/cljs was ignored.

123 #! this is ignored
;;=> 123
