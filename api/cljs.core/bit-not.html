bit-not

function/macrosince v0.0-927 clojure.core/bit-notEdit
(bit-not x)

Details:

Bitwise complement. Same as ~x in JavaScript.

Examples:

Bits can be entered using radix notation:

(bit-not 2r1100)
;;=> -13

Same numbers in decimal:

(bit-not 12)
;;=> -13
