(+)

Details:

Returns the sum of nums.

(+) returns 0.

Examples:

(+)
;;=> 0

(+ 1)
;;=> 1

(+ -10)
;;=> -10

(+ 1 2)
;;=> 3

(+ 1 2 3)
;;=> 6
