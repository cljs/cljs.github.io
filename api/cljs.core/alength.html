alength

function/macroAdded in 0.0-927 clojure.core/alength
(alength a)

For interop, it returns the length of a JavaScript array or string.

Examples:

(def a #js [1 2 3])

(alength a)
;;=> 3

(.-length a)
;;=> 3

(aget a "length")
;;=> 3

(count a)
;;=> 3

Source docstring:
Returns the length of the array. Works on arrays of all types.
