boolean

functionAdded in 0.0-927 clojure.core/booleanEdit
(boolean x)

Details:

Return false if x is false or nil. Otherwise return true. This is the truthiness condition used by if expressions.

Examples:

(boolean 1)
;;=> true

(boolean 0)
;;=> true

(boolean nil)
;;=> false

(defn ^boolean boolean
  [x]
  (cond
    (nil? x) false
    (false? x) false
    :else true))
