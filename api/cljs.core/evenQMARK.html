(even? n)

Details:

Returns true if n is an even number.

Throws an exception if n is not an integer.

(defn ^boolean even?
   [n] (if (integer? n)
        (zero? (bit-and n 1))
        (throw (js/Error. (str "Argument must be an integer: " n)))))
