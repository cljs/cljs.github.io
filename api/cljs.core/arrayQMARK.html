array?

functionsince v0.0-1798Edit
(array? x)

Details:

Returns true if x is a JavaScript array, false otherwise.

Examples:

(array? #js [1 2 3])
;;=> true

(array? [1 2 3])
;;=> false

(array? "hi")
;;=> false

(defn ^boolean array?
  [x]
  (if (identical? *target* "nodejs")
    (.isArray js/Array x)
    (instance? js/Array x)))
