drop-last

functionAdded in 0.0-927 clojure.core/drop-last
(drop-last s)
(drop-last n s)

Details:

Return a lazy sequence of all but the last n items in s.

n defaults to 1.

(defn drop-last
  ([s] (drop-last 1 s))
  ([n s] (map (fn [x _] x) s (drop n s))))
