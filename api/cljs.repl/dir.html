cljs.repl/dir

macroAdded in 0.0-2985 clojure.repl/dir
(dir ns)

Details:

Prints a sorted directory of public vars in a namespace ns.

Examples:

(require 'clojure.set)

(dir clojure.set)
;; Prints:
;;  difference
;;  index
;;  intersection
;;  join
;;  map-invert
;;  project
;;  rename
;;  rename-keys
;;  select
;;  subset?
;;  superset?
;;  union
;;
;;=> nil

(defmacro dir
  [ns]
  `(doseq [sym# (quote ~(sort (keys (ana-api/ns-publics ns))))]
     (println sym#)))
