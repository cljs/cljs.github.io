:browser-repl

compiler optionsince v1.9.183Edit

Preload code necessary for connecting to the natively supported browser REPL. Defaults to false.

:browser-repl true

Details:

When true, adds clojure.browser.repl/preload to :preloads.

The natively supported browser REPL must be started from a custom script in Clojure. Start the REPL by calling repl with the result of repl-env:

;; ClojureScript compiler script (in Clojure)

(require 'cljs.repl)
(require 'cljs.repl.browser)

(cljs.repl/repl (cljs.repl.browser/repl-env)
  :watch "src"
  :output-dir "out")
