cljs-dependents-for-macro-namespaces - function
more details >
(cljs-dependents-for-macro-namespaces namespaces)
(cljs-dependents-for-macro-namespaces state namespaces)
Takes a list of Clojure (.clj) namespaces that define macros and returns a list ClojureScript (.cljs) namespaces that depend on those macro namespaces. For example where example.macros is defined in the clojure file "example/macros.clj" and both 'example.core and 'example.util are ClojureScript namespaces that require and use the macros from 'example.macros : (cljs-dependents-for-macro-namespaces 'example.macros) -> ('example.core 'example.util)
