[org.clojure/clojurescript "0.0-2156"]

  • specify! added
  • tools.reader 0.8.3
  • :elide-asserts compiler option

  • analyzer/compiler/closure/repls did not properly close files
  • undeclared namespace warning regression
  • CLJS-751: reducers bug
