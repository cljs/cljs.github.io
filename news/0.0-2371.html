Oct 10, 2014 by David Nolen on Google Groups

This release includes minor fixes, faster collection iterators, and stays in sync with the latest transducers changes in Clojure master.

  • CLJS-866: Faulty ns macro desugaring
  • CLJS-869: When preamble is not found in source directory, compiler does not report it
